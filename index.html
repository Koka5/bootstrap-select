Works with .form-inline, .form-horizontal, and .form-vertical. Requires jQuery v1.7.2 and Bootstrap's Dropdown plugin.

Installation

Clone the repo and copy js/bootstrap-select.js and css/bootstrap-select.css into your project.

$> git clone git@github.com:jimjh/bootstrap-select.git
$> cp bootstrap-select/js/bootstrap-select.js path/to/your/project/js
$> cp bootstrap-select/css/bootstrap-select.css path/to/your/project/css

A minimal example is available in the repo for your reference.

Usage

Using Bootstrap's .prepend as an example, the boilerplate markup is as follows:

{% include boilerplate.md %}
  • data-select is used to trigger dropSelect() on the div element. You can omit that attribute and invoke the method manually.
  • dropdown-display is used to contain the displayed text.
  • dropdown-field is used to store the value of the selected option. Its value will be populated with the data-value attribute of the selected <a> element
